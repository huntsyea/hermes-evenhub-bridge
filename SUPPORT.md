# Support

Use GitHub issues for reproducible bugs, focused feature requests, and documentation
problems. Include the Hermes version, plugin version, operating system, and relevant
logs from `hermes logs` when reporting runtime behavior.

Use GitHub Security Advisories for vulnerabilities or anything involving token exposure,
unauthorized access, command execution, or signed sidecar distribution.

This project does not provide private operational support. Questions that are not bugs
may still be opened as issues when they are specific enough to improve the public docs.
